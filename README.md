# 🧭 导航站

一个轻量的**个人书签导航**，把你浏览器里的收藏夹变成一个漂亮的网格起始页。

🌐 在线访问：**[leilaomi.sld.tw](https://leilaomi.sld.tw)**

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 📂 **一键导入** | 拖入或选择浏览器导出的书签 HTML / 之前导出的 JSON |
| 🗂 **保留分类** | 完整保留浏览器里的文件夹结构，顶栏可切换 |
| ⭐ **常用** | 自动按点击次数推荐 Top 12 |
| 🔍 **跨分类搜索** | 名称 + URL 都参与匹配，带防抖 |
| 📤 **导出 JSON** | 数据备份 / 换设备使用 |
| 🌓 **三态主题** | 跟随系统 / 浅色 / 深色，自动记忆 |
| 🌐 **稳定图标** | 走 DuckDuckGo（国内可访问），失败回退首字母 |
| 📱 **PWA** | 可"添加到主屏幕"当 App 用 · 离线可访问 |
| ⌨️ **快捷键** | <kbd>/</kbd> 聚焦搜索 · <kbd>Esc</kbd> 清空 |
| 🎯 **拖放导入** | 直接把书签文件拖到页面上 |

## 🚀 使用

1. **导出书签**：Chrome / Edge → 收藏夹管理器 → 右上角菜单 → 导出书签
2. **打开网站** → 把文件拖进去（或点 📂 按钮选择）
3. 完成！数据保存在浏览器本地，不上传任何服务器

下次换设备或清缓存前，记得点 📤 **导出 JSON** 备份。

## 🛠 技术

- 纯 HTML/CSS/JS，**单文件**部署
- 无构建、无依赖、无后端
- 全部数据 `localStorage`，**不上传任何服务器**
- 标准 Netscape 书签格式解析（兼容 Chrome / Edge / Firefox / Safari）

## 📦 部署

仓库内容直接传到任何静态托管：

- **GitHub Pages**（当前）：push `main` 自动部署
- **Cloudflare Pages / Vercel / Netlify**：导入仓库一键发布
- **自己服务器**：4 个核心文件丢到 web 根目录即可

```
index.html      # 主页面
manifest.json   # PWA 配置
sw.js           # Service Worker
icon.svg        # 图标
```

## 🎨 自定义主题色

打开 `index.html`，在 `:root` 改色：

```css
:root{
  --accent:#4a90c4;     /* 主色调 */
  --accent-bg:#e8f1f9;  /* hover 浅背景 */
}
```

## ⌨️ 快捷键

| 键 | 作用 |
|------|------|
| <kbd>/</kbd> 或 <kbd>s</kbd> | 聚焦搜索框 |
| <kbd>Esc</kbd> | 清空搜索 |
